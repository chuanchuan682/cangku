## 📋 项目整体规划

### 一、项目概述

**项目名称**：MBTI 16型人格测试（iOS风格版）

**核心目标**：做一个纯前端、单页面的MBTI性格测试网站，测试题与标准MBTI一致，完成后展示详细的结果解读，界面完全模仿iOS风格，页面切换动画丝滑流畅。

**技术方案**：纯前端（HTML + CSS + JavaScript），无需后端服务器，打开即用。

---

### 二、技术栈选型

| 类别 | 技术 | 说明 |
|------|------|------|
| 核心语言 | HTML5 + CSS3 + JavaScript | 纯前端，无需框架 |
| UI组件库 | **ios-style-ui** | 零依赖，完美复刻iOS 17+风格 |
| 动画引擎 | CSS Transition + Web Animations API | 原生实现丝滑过渡 |
| 数据存储 | 纯JS对象/数组 | 所有题目和结果数据硬编码 |

**为什么这样选**：
- 纯前端方案对编程小白最友好，无需配置服务器、数据库
- `ios-style-ui` 能一键获得iPhone风格的按钮、卡片、导航栏等组件
- 原生CSS动画配合 `requestAnimationFrame` 能实现60fps的丝滑体验

### 三、功能模块规划

#### 模块1：首页（欢迎页）
- iOS风格大标题：“MBTI性格测试”

### 六、标准MBTI题目来源

93道标准MBTI题目分为四个部分：

| 部分 | 题号 | 测量维度 |
|------|------|----------|
| 第一部分 | 1-26 | E/I（外向/内向） |
| 第二部分 | 27-52 | S/N（感觉/直觉） |
| 第三部分 | 53-78 | T/F（思考/情感） |
| 第四部分 | 79-93 | J/P（判断/感知） |

**典型题目示例**：
- “你倾向从何处得到力量？” → (E)别人 / (I)自己的想法
- “你倾向通过以下哪种方式收集信息？” → (N)对可能发生之事的想像 / (S)对目前状况的实际认知
- “你倾向如此做决定？” → (F)首先依你的心意 / (T)首先依你的逻辑
- “处理许多事情上，你会喜欢？” → (P)凭兴所至行事 / (J)按照计划行事

> ⚠️ **版权提醒**：标准93题MBTI是有版权的测评工具。个人项目用于学习和展示没问题，但公开发布或商业使用需注意版权合规。你也可以考虑缩减到60题或更少。
